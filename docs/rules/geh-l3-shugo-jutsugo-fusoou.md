## 書籍根拠

D 節「a) 内容の点検」3 内容点検の具体的事項 (11)。書籍は内容点検の項目として、主語と述語が照応しているか、また距離が離れすぎていないか（途中の説明が長すぎていないか）を確認するよう求めている。また同 (10) では係りと結びの乱れがないかを確認する項目も挙げている。

## 動作原理

**入力**: 一文（または接続が密な2〜3文のまとまり）  
**判断基準**:  
1. LLM が文の主語（「〜は」「〜が」）と述語を特定し、意味的に対応しているかを評価する。  
2. 「〜のは……ことだ」「〜のは……という点である」など、主語が名詞節・指示語の場合は述語との対応を確認する。  
3. 主語が複数の節にまたがった後に述語が現れる場合、途中の修飾節が長く距離が離れているケースを検出する。  
4. 主語に対応する意味的な述語類型（行為・状態・判断・感情等）と実際の述語が噛み合わない場合に指摘する。

**出力**: 不照応と判断された文、具体的にどの主語と述語が噛み合っていないかの説明、可能であれば修正案の提示。

## なぜ L1/L2 で不可能か

- 主語・述語の対応は文節間の意味的適合性の判断であり、形態素レベルの品詞タグでは取得できない。
- 特に主語が複数の節を跨いだ後に述語が現れる場合、どの主語に述語が掛かるかの解析には文全体の統語・意味理解が必要。
- 「係りと結びの乱れ」は句・節ごとのスコープを理解しないと検出不可能で、正規表現では構造を把握できない。
- L2（形態素解析）は品詞境界まで取れるが、「重要なのは/問題だと思われる」のような名詞節述語の意味的不一致は判定できない。

## 例

### positive（指摘なし）

```
ここで重要なのは、実験条件を一定に保つという点である。
```
（「重要なのは」→「点である」で照応している）

```
今回の課題は、開発コストを削減することだ。
```
（主語「課題は」→述語「削減することだ」が適切に対応）

### negative（不照応として指摘）

```
ここで重要なのは、実験条件を一定に保つ必要がある。
```
（「重要なのは」という主語に対し、「必要がある」では呼応が乱れている。「重要なのは〜という点だ」または「実験条件を一定に保つ必要がある」に整理すべき）

```
私たちの研究の目的は、新手法の有効性を確認し、従来手法との比較実験を行い、精度向上が期待できる。
```
（「目的は」に対して述語「期待できる」は照応せず、文が破綻している）

## 想定される偽陽性・運用上の注意

- **文学的倒置・強調表現**: 意図的に主語と述語を離した修辞技法は誤検出しやすい。novel モードでは適用しないことを推奨する。
- **複文の接続節**: 従属節内の主語と主節の述語を誤って不照応と判定するリスクがある。LLM は主節の主語を正しく特定することが前提。
- **主語の省略**: 日本語は主語を省略する慣行が強く、LLM が省略された主語を正しく補完できない場合に誤検出が発生する。
- severity は warning に留め、編集者による確認を前提とする。
