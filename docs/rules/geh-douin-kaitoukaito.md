# geh-douin-kaitoukaito — 「回答」と「解答」の混用検出

- **ruleId**: `geh-douin-kaitoukaito`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 全モード（novel / official / blog / academic / sns）

## 意図

どちらも「かいとう」と読む「回答」と「解答」は混同しやすい。
原稿編集 第2版 E 節 8 項は同音・同訓異義語の誤用を点検するよう求めている。

- **回答**: 質問・依頼に対する返事・返答（アンケートへの回答・照会への回答）
- **解答**: 問題・試験に対する答え（解答欄・模範解答・解答用紙）

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 8 同音・同訓異義語

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `アンケートにご回答ください。` | 指摘なし |
| 誤例 | `アンケートにご解答ください。` | `解答` を `回答` に修正 |

## 偽陽性回避

- 「アンケート/調査票/質問票/フォーム + に + ご解答/解答」の連語パターンのみ検出。
- 「試験の解答」「模範解答」「解答欄」などは対象外（正用のため）。
- 単独の「解答」は誤用とは言えないため除外している。

## 偽陽性リスク

**低**: 連語パターンで文脈を限定しており、「解答欄」「模範解答」などの正用とは明確に区別できる。
