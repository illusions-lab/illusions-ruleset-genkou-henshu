# geh-hojo-kanji — 補助的用法の語の漢字表記検出

- **ruleId**: `geh-hojo-kanji`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: official / academic（公用文・学術文書向け）

## 意図

原稿編集 第2版 E 節 6 項「仮名書きが望ましい語」では、公用文の方針を引用しつつ、
形式名詞・補助動詞等で補助的に用いられる語は仮名書きにすることが示されている。
具体的には以下のような語句が仮名書きの対象として列挙されている:

- `こと`（「許可しないことがある」）
- `とき`（「事故のときは連絡する」）
- `わけ`（「賛成するわけにはいかない」）
- `〜てあげる`（補助動詞「あげる」）

これらが漢字で書かれた場合（「許可しない事がある」等）を検出する。

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 6 仮名書きが望ましい語

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `許可しないことがある。` | 指摘なし |
| 誤例 | `許可しない事がある。` | `事` を `こと` に変更提案 |
| 誤例 | `賛成する訳にはいかない。` | `訳` を `わけ` に変更提案 |

## 偽陽性回避

- 動詞（「する・ない・ある・できる・いる・なる」等）の直後の形式名詞のみ対象とし、
  語頭や漢語複合語の一部（「事件」「事実」「事態」「事業」等）は除外する。
- 「事件」「事実」「事態」等の複合語は直後に漢字が続くため先読みの助詞条件で除外される。
- 実質名詞用法（「その事は忘れた」など完全な名詞用法）は偽陽性が生じる可能性がある。

## 偽陽性リスク

**中**: 形式名詞と実質名詞の区別は文脈依存のため、100%の精度はL1では難しい。
「その事がある」のような実質名詞的用法を誤検出する可能性がある。
該当ルールは官公庁・学術文書モードに限定し、小説・ブログ等では無効とする。
