# geh-keishiki-meishi-l2 — 形式名詞の仮名書き検出（形態素解析）

- **ruleId**: `geh-keishiki-meishi-l2`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: official / academic（公用文・学術文書向け）

## 意図

原稿編集 第2版 E 節 6 項「仮名書きが望ましい語」では、
形式名詞として用いる語は仮名書きにするとして次の用例が列挙されている：

- `こと`（許可しないことがある。）
- `とき`（事故のときは連絡する。）
- `ところ`（現在のところ差し支えない。）
- `もの`（正しいものと認める。）
- `わけ`（賛成するわけにはいかない。）

これらの語が漢字形（事・時・所・物・訳）で書かれており、
かつ形式名詞として用いられている場合に検出し、仮名書きへの変更を提案する。

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 6 仮名書きが望ましい語

## 形態素解析による判定方法

kuromoji は形式名詞（補助的な意味で使われる名詞）を次のタグで識別する：

| フィールド | 値 |
| ---------- | --- |
| `pos` | `名詞` |
| `pos_detail_1` | `非自立` |

実質名詞（独立した具体的意味を持つ名詞）は `pos_detail_1` が `一般` / `固有名詞` / `数` 等となるため、
`非自立` タグを使うことで形式名詞用法のみを確実に絞り込める。

## 検出対象マッピング

| 漢字形 | 仮名推奨形 |
| ------ | ---------- |
| 事 | こと |
| 時 | とき |
| 所 | ところ |
| 物 | もの |
| 訳 | わけ |

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `許可しないことがある。` | 指摘なし |
| 誤例 | `許可しない事がある。` | `事` を `こと` に変更提案 |
| 誤例 | `事故のときは連絡する。` | 指摘なし（既に仮名） |
| 誤例 | `事故の時は連絡する。` | `時` を `とき` に変更提案 |
| 誤例 | `正しい物と認める。` | `物` を `もの` に変更提案 |
| 誤例 | `賛成する訳にはいかない。` | `訳` を `わけ` に変更提案 |

## 偽陽性回避

- `pos_detail_1 === "非自立"` のトークンのみを対象とし、実質名詞との区別を形態素解析に委ねる。
- `事件`（じけん）・`時間`（じかん）・`場所`（ばしょ）・`物語`（ものがたり）のような複合語は、
  kuromoji が別のトークンとして解析し `pos_detail_1` が `一般` 等になるため除外される。
- 既に仮名書きされているトークン（surface が推奨形と一致）はスキップする。

## L1 ルール geh-hojo-kanji との関係

| 観点 | geh-hojo-kanji (L1) | geh-keishiki-meishi-l2 (L2) |
| ---- | ------------------- | --------------------------- |
| 検出方法 | 正規表現 + 前後文脈パターン | 形態素解析 `名詞-非自立` タグ |
| FP リスク | 前後の助詞パターンに依存するため中程度 | 形態素解析精度に依存するが低〜中 |
| 適用条件 | 実行コスト低 | kuromoji 実行が必要 |
| 補完効果 | 形態素解析不要な環境でも動作 | 文脈パターンで取りこぼした箇所を補完 |

## 偽陽性リスク

**低〜中**: 形態素解析器（kuromoji）の精度に依存する。
稀に実質名詞が `非自立` とタグ付けされる場合があるが、対象語が限定的（5語）なため
実際の誤検出は少ない。公用文・学術文書モードに限定し、小説・ブログ等では無効とする。
