# geh-l3-douin-ippan — 同音・同訓異義語の文意判別（汎用）

> ⚠️ ステータス: L3（LLM支援）・未実装。SDK が L3 を支持した後に実装予定。本ドキュメントは仕様の先行定義です。

- **ruleId**: `geh-l3-douin-ippan`
- **nameJa**: 同音・同訓異義語の文意による誤用検出（汎用）
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L3（LLM支援）
- **想定 severity**: warning
- **適用モード案**: 全モード（novel / official / blog / academic / sns）

## 書籍根拠

E 節「b) 表記整理の具体的事項」8 同音・同訓異義語。書籍はデジタル原稿で仮名漢字変換を使う場合に同音・同訓異義語の誤りが生じやすく、使い分けに注意するよう求めている。例示として「対照⟷対象、異議⟷異義・意義、異状⟷異常、回答⟷解答、解放⟷開放、過程⟷課程、小数⟷少数、史料⟷資料、実体⟷実態、大系⟷体系、野生⟷野性」など多数を挙げている。このうち固定的な連語で検出できるペアは既存の L1 ルール（`geh-douin-*`）がカバーしているが、固定連語パターンに当てはまらない文脈依存の誤用は L3 が担当する。

## 動作原理

**入力**: 一文または段落  
**判断基準**:  
1. LLM が書籍の例示語リスト（および類似の同音・同訓ペア）を参照し、文中の漢字語がその文脈で意味的に適切かを判断する。  
2. 既存 L1 ルールでカバー済みの固定連語パターン（「研究対象」「比較対照」等）は除外し、それ以外の自由な文脈における誤用候補を検出する。  
3. 対象語の意味が文の意図と矛盾している場合に「誤用の可能性あり」として指摘し、正しいと考えられる語を提案する。  
4. 判断が拮抗する場合（文脈から確定できない場合）は「確認を推奨」のレベルで提示する。

**対象語の例（書籍例示に基づく）**:
- 異議（意見の相違）⟷ 異義（語義の違い）⟷ 意義（意味・価値）
- 異状（普通でない状態）⟷ 異常（通常から外れた状態・程度が大きい）
- 解放（束縛を解く）⟷ 開放（開け放つ・自由に使えるようにする）
- 小数（少ない数・数学用語）⟷ 少数（数が少ない・少数派）
- 史料（歴史研究の資料）⟷ 資料（一般の参考データ）
- 実体（本当の姿・体）⟷ 実態（実際の状態・ありさま）
- 大系（体系的な全集）⟷ 体系（組織立った系統）
- 野生（自然のままの生き物の状態）⟷ 野性（野性的な性質・気質）

**出力**: 誤用の可能性がある語、その位置、推奨する語と理由の簡潔な説明。

## なぜ L1/L2 で不可能か

- 既存 L1 ルール（`geh-douin-*`）は「研究対象」「比較対照」などの固定連語パターンに限定して誤用を検出する。しかし、同じペアでも「この議論の異議は明確だ」（異義の誤用）のように固定連語以外の文脈で誤用が現れた場合は検出できない。
- 「解放」と「開放」は文脈によって正用と誤用が逆転する（「囚人を解放する」は正用、「図書館を解放する」は「開放する」が正しい）。この区別は文の主語・目的語の意味カテゴリを理解する必要があり、正規表現では不可能。
- 形態素解析は読みと品詞を提供するが、同音語が「どちらの語義が意図されているか」は文全体の意味を解釈しなければ判断できない。

## 例

### positive（指摘なし）

```
歴史研究には一次史料の収集が不可欠だ。
```
（「史料」は歴史的資料の意で正しく使用されている）

```
社会の実態を調査するため、アンケートを実施した。
```
（「実態」は実際の状態・ありさまの意で文脈に合っている）

### negative（誤用の可能性として指摘）

```
歴史研究には一次資料の収集が不可欠だ。
```
（歴史研究の文脈では「史料」が適切。「資料」は一般的参考データを指すため誤用の可能性がある）

```
この政策の実体はまだ明らかになっていない。
```
（政策の実際の状態・ありさまを指す文脈では「実態」が適切。「実体」は本当の姿・本質的な存在を指す語）

## 想定される偽陽性・運用上の注意

- **専門分野の慣用**: 哲学・法律など分野によっては「実体」が技術用語として定着している（例：実体法）。分野情報を入力メタデータで与えることが望ましい。
- **意図的な使い分け**: 著者が同音語を意図的に使い分けている場合（たとえば「大系」を書名として使う）は正用。固有名詞・書名・引用内は除外する。
- **L1 ルールとの重複除外**: `geh-douin-taishoutaisho`、`geh-douin-kaitoukaito`、`geh-douin-kateikateii` がカバーする固定連語パターンはこのルールから明示的に除外し、二重指摘を防ぐ。
- 判断の確信度が低い場合は severity を info に下げて提示し、著者・編集者の確認を促す。
