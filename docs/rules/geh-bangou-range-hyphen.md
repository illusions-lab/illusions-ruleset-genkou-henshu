# geh-bangou-range-hyphen — 数値範囲のハイフン使用検出

- **ruleId**: `geh-bangou-range-hyphen`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 全モード（novel / official / blog / academic / sns）

## 意図

原稿編集 第2版 E 節 15 項「横組の数字表記 (6) 数の幅」には、
起点と終点の幅を表記する場合は「12-15ページ, 1895-1970のように二分ダーシを用いベタ組にする」
と明記されている（横組の場合。縦組は別途）。

この原則に従い、横組で数値・ページ範囲を半角ハイフン「-」で示している箇所を検出し、
全角ダーシ「—」または波形「〜」への統一を提案する。

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 15 横組の数字表記 (6) 数の幅

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `12〜15ページを参照のこと。` | 指摘なし |
| 正例 | `1895—1970年の出来事。` | 指摘なし |
| 誤例 | `12-15ページを参照のこと。` | `12-15` を `12〜15` に変更提案 |
| 誤例 | `1895-1970年の出来事。` | `1895-1970` を `1895〜1970` に変更提案 |

## 偽陽性回避

- 数字+ハイフン+数字の直後に単位語（ページ・頁・年・月・日・章・巻・行・cm・mm・kg など）が
  続く場合のみ検出する。単位語がない単純な計算式や型番（ISO-9001 等）は対象外。
- 電話番号（03-1234-5678 等）との誤検出を防ぐため、直後に和文の単位語が来るパターンに限定する。

## 偽陽性リスク

**低**: 直後に続く単位語での文脈限定により、電話番号や型番との誤検出は少ない。
欧文・英数混在文書ではページ表記に半角ハイフンを意図的に使う場合があるが、それは別途方針を定めること。
