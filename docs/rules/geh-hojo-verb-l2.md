# geh-hojo-verb-l2 — 補助動詞の仮名書き検出（形態素解析）

- **ruleId**: `geh-hojo-verb-l2`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L2（形態素解析）
- **既定**: 有効 / severity = warning
- **適用モード**: official / academic（公用文・学術文書向け）

## 意図

原稿編集 第2版 E 節 6 項「仮名書きが望ましい語」では、補助動詞は仮名書きにするとして
次の用例が列挙されている:

- `……ていく`（「負担が増えていく」）
- `……てくる`（「寒くなってくる」）
- `……ておく`（「通知しておく」）
- `……てみる`（「見てみる」）
- `……てしまう`（「書いてしまう」）
- `……てあげる`（「図書を貸してあげる」）
- `……ていただく`（「報告していただく」）

これらの用言が漢字形（「行く」「来る」「置く」「見る」「仕舞う」「上げる」「頂く」等）で
書かれた場合に検出する。

形態素解析（L2）を使う理由: 「行く」は本動詞（「図書館に行く」）としては漢字で問題なく、
補助動詞として用いる場合のみ仮名書きが推奨される。接続助詞「て/で」の直後に来る場合のみを
補助動詞用法と判定することで、本動詞との区別を行う。

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 6 仮名書きが望ましい語

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `負担が増えていく。` | 指摘なし |
| 誤例 | `負担が増えて行く。` | `行く` を `いく` に変更提案 |
| 誤例 | `書いて仕舞う。` | `仕舞う` を `しまう` に変更提案 |

## 偽陽性回避

- 接続助詞「て/で」の直後に来る動詞のみを対象とし、本動詞用法は除外する。
- 対象の動詞の基本形（`basic_form`）が補助動詞辞書に含まれる場合のみ検出する。
- 既に仮名書きされている場合（surface が推奨形と一致）はスキップする。

## 偽陽性リスク

**低〜中**: 形態素解析器の精度に依存する。
接続助詞「て」の判定が誤った場合は偽陽性が生じる可能性がある。
官公庁・学術文書モードに限定し、小説・ブログ等では無効とする。
