# geh-douin-kateikateii — 「過程」と「課程」の混用検出

- **ruleId**: `geh-douin-kateikateii`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 全モード（novel / official / blog / academic / sns）

## 意図

どちらも「かてい」と読む「過程」と「課程」は混同しやすい。
原稿編集 第2版 E 節 8 項は同音・同訓異義語の誤用を点検するよう求めており、
「過程」と「課程」が原文中の誤用例として挙げられている。

- **過程**: 物事が進行・変化する経緯・プロセス（開発の過程・成長の過程）
- **課程**: 一定の教育・学習の区分（博士課程・教育課程・学習課程）

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 8 同音・同訓異義語

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `開発の過程で新たな発見があった。` | 指摘なし |
| 誤例 | `開発の課程で新たな発見があった。` | `課程` を `過程` に修正 |

## 偽陽性回避

- 「開発/制作/製作/研究/進行/作業/検討 + の + 課程」という連語パターンのみ検出。
- 「博士課程」「教育課程」等の語は直後に「修了/終了/入学/在学/卒業」等が続く場合は除外（先読みで制限）。
- 「課程に在学」「課程を修了」のような正用は対象外。

## 偽陽性リスク

**低**: 物事の進行を表す動詞・名詞の直後の「課程」に限定しており、教育・学習の文脈での「課程」は明示的に除外している。
