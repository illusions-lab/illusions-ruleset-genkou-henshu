# geh-douin-taishoutaisho — 「対照」と「対象」の混用検出

- **ruleId**: `geh-douin-taishoutaisho`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 全モード（novel / official / blog / academic / sns）

## 意図

仮名漢字変換ではどちらも「たいしょう」と読む「対照」と「対象」は混同しやすい。
原稿編集 第2版 E 節 8 項は同音・同訓異義語の誤用を点検するよう求めており、
この2語は典型的な誤用例として原文に挙げられている。

- **対照**: 二つのものを照らし合わせて比べること（比較対照・対照表・対照実験）
- **対象**: ある行為・考えが向けられる物や人（研究対象・対象者・支援対象）

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 8 同音・同訓異義語

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `今回の研究対象は成人男性に限定した。` | 指摘なし |
| 誤例 | `今回の研究対照は成人男性に限定した。` | `対照` を `対象` に修正 |
| 誤例 | `対照者を募集する。` | `対照者` を `対象者` に修正 |

## 偽陽性回避

- 「対照実験」「比較対照」「対照表」「対照的」「対照群」は正用のため除外（先読みで制限）。
- 「研究/調査/支援 + の + 対照」の固定連語パターンのみ検出し、単独の「対照」は原則対象外。
- 統計・心理学分野では「対照群（control group）」が正用。パターンで除外済み。

## 偽陽性リスク

**低**: 固定的な連語パターンに限定しており、「対照実験」「対照表」等の正用は明示的に除外している。
「対照者」は一般文書では「対象者」の誤用がほとんどだが、ごく稀に統計の専門文書で正用として使われる場合がある。
