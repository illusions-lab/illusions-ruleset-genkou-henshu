# geh-katakana-trailing-choon — 片仮名語末尾の長音符省略検出

- **ruleId**: `geh-katakana-trailing-choon`
- **ガイドライン**: 原稿編集 第2版 (`genkou-henshu-2`)
- **レベル**: L1（正規表現）
- **既定**: 有効 / severity = warning
- **適用モード**: 全モード（novel / official / blog / academic / sns）

## 意図

外来語の表記（1991年内閣告示第2号）の留意事項（細則的な事項）には、
英語の語末 -er/-or/-ar などに当たるものは原則としてア列の長音とし
長音符号「ー」を用いて書き表すと定められている（ただし慣用に応じて省くことができる）。

原稿編集 第2版 E 節 12 項は、外来語表記は「不統一にならないように整理する」ことを求める。
長音省略形（「コンピュータ」）と長音付き形（「コンピューター」）が同一文書に混在すると
不統一となるため、省略形を検出して長音付き形を提案する。

## 出典

原稿編集 第2版（日本エディタースクール）E 表記の整理 b) 表記整理の具体的事項 12 外来語の表記
（内閣告示「外来語の表記」1991年 留意事項 細則 ID-3 参照）

## 例

| 区分 | テキスト | 期待 |
| ---- | -------- | ---- |
| 正例 | `コンピューターで文書を作成する。` | 指摘なし |
| 誤例 | `コンピュータで文書を作成する。` | `コンピュータ` を `コンピューター` に統一提案 |
| 誤例 | `プリンタを接続した。` | `プリンタ` を `プリンター` に統一提案 |

## 対象語リスト

コンピュータ・プリンタ・スキャナ・モニタ・センサ・ルータ・サーバ・フォルダ・カレンダ・アダプタ・シリンダ

## 偽陽性回避

- 直後に「ー」が続く場合（既に長音付き）は先読みで除外。
- 対象語は代表的な混用例のみに限定しており、意図的な省略形（規格名・固有名詞等）は個別対応が必要。

## 偽陽性リスク

**低**: 直後の「ー」有無で確実に判別可能。対象語は固定リストであり副作用は少ない。
分野・スタイルによっては省略形を意図的に採用している場合があるが、警告（warning）として提示し最終判断は執筆者に委ねる。
